# User_config-manager
A simple python user configuration manager using JSON
A simple python project that demostrate how to :
-load user settings
-save setting to a Json file
- update configuration dynamically

  ## technologies
  -python
  -json

  ## how to run
  - clone or download the repository
  - run 'config_manager.py'
  - the app will create and update 'config.json' automatically
